A Mini Web Server Implementation in Java:
1. Can accept http requests and send out formatted responses
2. Can set environment through CGI
3. Authentication through .htpasswd also implemented for protected files.

To run the project from the command line, go to the dist folder and
type the following:

java -jar "WebServerProject.jar" 

Test Files are located in ./public_html for the public domain and ./private for the private domain.
